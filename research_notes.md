# AI沙龙知识卡片系统调研笔记

## 1. MD2Card 工具
- 免费、无需登录、无限生成
- 支持Markdown转知识卡片，20+主题风格
- 支持API调用、MCP Server、Coze插件
- 支持长文自动拆分、实时预览、一键导出
- 可私有部署（Docker）
- 工作流：Markdown内容 → MD2Card → 精美知识卡片图片

## 2. Coze（扣子）工作流方案
- 输入知识主题 → 大模型解读生成文案 → 生图模型生成手绘知识卡片 → 上传飞书表格
- 使用Nano Banana Pro生图模型（中文文字处理能力强）
- 可与飞书多维表格集成，自动存储
- 完整工作流节点：科普解说→绘图提示词→图片生成→循环查询→上传飞书

## 3. 科大讯飞录音笔
- 录音转文字后可智能提炼纪要
- 搭载讯飞星火大模型
- 可生成思维导图
- 讯飞听见平台支持音视频转写

## 4. 飞书生态
- 飞书AI智能伙伴：会中AI总结、智能调研
- 飞书多维表格：可接入大模型进行信息提取、分类、总结
- 飞书CLI（MCP）：连接AI到飞书文档、表格、日历等
- 飞书开放平台API：可读取飞书文档内容

## 5. 其他知识卡片工具
- MemoleCard 魔卡：杂乱文字→高颜值知识卡片
- WiseMind AI：本地知识库+智能笔记+知识卡片+海报生成
- reminds：AI卡片笔记，记连忆用四大哲学
- Mo卡片：AIGC学习工具，卡片式学习

## 6. 用户场景分析
- 活动前：从飞书等平台提炼AI大神思路 → 生成引题卡片
- 活动中：科大讯飞录音笔全程记录
- 活动后：录音转文字 → 提炼知识点 → 生成分享卡片/对话框
- 知识积累：卡片归档、分类、回顾
